Movie Finder App
This is a mobile movie application built using React Native and Expo. The app allows users to explore movies, search for their favorite titles, and view popular movies based on user searches.

The application fetches movie data from an external movie API and displays it in a clean and user-friendly interface. It is designed to be fast, responsive, and easy to use on mobile devices.
